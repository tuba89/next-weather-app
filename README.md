## ğŸš€ Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/tuba89/next-weather-app.git
   cd next-weather-app
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**

   ```env
   OPENWEATHER_API_KEY=your_api_key_here
   ```

